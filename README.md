\# Autolysis – Automated CSV Data Analysis



Project Description:

This project is a Python script that automatically analyzes a CSV file.

It shows basic information, checks missing values, and creates graphs.



Tools Used:

\- Python

\- Pandas

\- Matplotlib

\- uv



Files in this Project:

\- autolysis.py  → main Python script

\- happiness.csv → dataset file

\- test.csv      → small test dataset

\- \*.png         → output graphs

\- README.md     → project explanation



How to Run the Project:

1\. Open Command Prompt

2\. Go to project folder

3\. Run the command:

&nbsp;  uv run autolysis.py happiness.csv



What the Script Does:

\- Loads the CSV file

\- Prints number of rows and columns

\- Shows missing values per column

\- Displays summary statistics

\- Saves histograms as PNG files



Output:

\- Console summary

\- Histogram images saved in the folder
